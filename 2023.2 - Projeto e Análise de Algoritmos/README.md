## Destaque: [Busca em Grafo](./Atividades/6.%20Busca%20em%20Grafo/)

- **Problema:**  
Calcular a quantidade de caminhos distintos de um vértice origem até todos os demais vértices do grafo.

- **Solução encontrada:**  
Implementação de uma busca em largura (BFS) para percorrer o grafo, através uma fila de adjacências.  
Para cada vértice removido da fila, é executada uma busca em profundidade (DFS) que, recursivamente, conta todos os caminhos possíveis a partir dele.
